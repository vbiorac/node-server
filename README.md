# Flaschenpost webshop server
Node webserver for Flaschenpost webshop

## Getting started

### Installation

Clone repository
```shell
git clone git@bitbucket.org:flaschenpost_dev/webshop-server.git
```

Change directory
```shell
cd webshop-server
```

Install packages
```shell
yarn install
```

## Usage

### Development

To run development server, run command
```shell
yarn dev
```

### Test

To run tests, run command
```shell
yarn test
```
